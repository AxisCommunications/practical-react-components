# Select

Select components are used for collecting user provided information from a
list of options inside a dropdown menu.

## Basic usage

### Without label

```typescript type=live
<Select
  value="apple"
  options={[
    { value: 'apple', label: 'Apple' },
    { value: 'banana', label: 'Banana' },
    { value: 'pie', label: 'Pie', disabled: true },
  ]}
  onChange={() => {}}
  placeholder="Select..."
  width="small"
/>
```

### With label

```typescript type=live
<div>
  <SelectField
    label="Select field label"
    value="apple"
    options={[
      { value: 'apple', label: 'Apple' },
      { value: 'banana', label: 'Banana' },
      { value: 'pie', label: 'Pie', disabled: true },
    ]}
    onChange={() => {}}
    placeholder="Select..."
    width="small"
  />
</div>
```

### Combination of some Select fields

If you want to use several Select fields, you can use `FormSection` component
which is in `Content` folder. That gives section title(which is optional) and
gives proper margin to all children's bottom. `SpaceBlock` which is in `Content`
folder is useful for adjusting space between each `FormSection`.

```typescript type=live
<>
  <FormSection title="FormSection title">
    <SelectField
      label="Select field label which is very very long"
      value="apple"
      options={[
        { value: 'apple', label: 'Apple' },
        { value: 'banana', label: 'Banana' },
        { value: 'pie', label: 'Pie', disabled: true },
      ]}
      onChange={() => {}}
      placeholder="Select..."
      width="small"
    />
    <SelectField
      label="Select field label which is very very long"
      value="apple"
      options={[
        { value: 'apple', label: 'Apple' },
        { value: 'banana', label: 'Banana' },
        { value: 'pie', label: 'Pie', disabled: true },
      ]}
      onChange={() => {}}
      placeholder="Select..."
      width="small"
    />
  </FormSection>
  <SpaceBlock variant={32} />
  <FormSection title="Second FormSection title">
    <SelectField
      label="Select field label which is very very long"
      value="apple"
      options={[
        { value: 'apple', label: 'Apple' },
        { value: 'banana', label: 'Banana' },
        { value: 'pie', label: 'Pie', disabled: true },
      ]}
      onChange={() => {}}
      placeholder="Select..."
      width="small"
    />
    <SelectField
      label="Select field label which is very very long"
      value="apple"
      options={[
        { value: 'apple', label: 'Apple' },
        { value: 'banana', label: 'Banana' },
        { value: 'pie', label: 'Pie', disabled: true },
      ]}
      onChange={() => {}}
      placeholder="Select..."
      width="small"
    />
  </FormSection>
</>
```

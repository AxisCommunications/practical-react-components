# TextArea

The textarea tag defines a multi-line text input control. A text area can hold an unlimited number of characters.

## Basic usage

### Without label

```typescript type=live
<TextArea value="" onValueChange={() => {}} placeholder="Placeholder" />
```

### With label

```typescript type=live
<TextAreaField
  label="TextArea label"
  value=""
  onValueChange={() => {}}
  placeholder="Placeholder"
/>
```

## Props

TextArea forwards all unknown props to the base native `<textarea>` component. See
[`<textarea>` on MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/textarea) for more
details on which additional props that are available.

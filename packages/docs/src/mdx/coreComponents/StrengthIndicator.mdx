# StrengthIndicator

A StrengthIndicator can signify the relative strength of a piece of data or information. Some projects use it as a
password strength indicator together with DropBox [zxcvbn](https://github.com/dropbox/zxcvbn) library.

## Basic usage

```typescript type=live
<StrengthIndicator strength={0.5} helpText="ok" />
```

## Props

[`<div>` on MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/div) for more
details on which additional props that are available.

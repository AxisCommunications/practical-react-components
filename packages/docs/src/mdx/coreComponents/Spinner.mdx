# Spinner

A spinner can be used to indicate that something is loading.
Use this when loading takes a longer time (too long for a skeleton loader),
but not so long that you would need a progress indicator.

## Demo

<div
  style={{
    padding: '2px',
    display: 'inline-grid',

    gridTemplateColumns: 'repeat(auto-fit, minmax(100px, 1fr))',
    width: '100%',
  }}
>
  <Spinner type="primary" label="primary" />
  <Spinner type="dashed" label="dashed" />
</div>

## Basic usage

```typescript type=live
<Spinner type="primary" label="label" />
```

## Props

Spinner forwards all unknown props to the base native `<div>` component. See
[`<div>` on MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/div) for more
details on which additional props that are available.

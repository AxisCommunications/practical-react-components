# RadioButton

## RadioButton

Radio buttons allow the user to select one option from a set.

Use radio buttons when the user needs to see all available options.
If available options can be collapsed, consider using a dropdown menu because it uses less space.

Radio buttons should have the most commonly used option selected by default.

RadioButtonGroup is a helpful wrapper used to group Radio components that provides an easier API,
and proper keyboard accessibility to the group.

## RadioIconButton

A RadioIconButton is a clickable element which will execute a command when clicked.
The commands purpose is stored in a function which is then sent to the RadioIconButton's 'onClick' property upon being clicked.

## RadioButton Props

RadioButton forwards all unknown props to the base native `<input>` component.
See [`<input type="checkbox">` on MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input/checkbox)
for more details on which additional props that are available.

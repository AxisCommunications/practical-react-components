# SimpleTable

A simple table based on a regular HTML `<table>`. Column size is determined automatically
by the browser, and cannot be modified by the user.
If checkboxes/menus are needed, you need to add these yourself as separate columns.
For more advanced functionality and built-in checkbox/menu, please use the `Table` component instead.

export const DemoComponent = ({}) => {
  const clickFunc = useCallback(id => {
    window.alert(`You clicked on: ${id}`)
  }, [])
  return (
    <Wrapper>
      <GlobalScrollbarStyle />
      <SimpleTable
        maxHeight={3}
        header={TABLE_HEADER_DATA.map(({ title }, id) => (
          <TableHeaderText key={id}>{title}</TableHeaderText>
        ))}
        className="my-simple-table"
      >
        {DEVICE_LIST.map((device, index) => (
          <SimpleTableRow
            key={device.serialNumber}
            disabled={index === 2}
            className="my-simple-table-row"
            onClick={
              index === 5 ? () => clickFunc(device.serialNumber) : undefined
            }
          >
            <Typography>{device.serialNumber}</Typography>
            <Typography>{device.name}</Typography>
            <Typography>{device.ip}</Typography>
          </SimpleTableRow>
        ))}
      </SimpleTable>
    </Wrapper>
  )
}

## Table Props

SimpleTable forwards all unknown props to the base native `<table>` component. See
[`<table>` on MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/table) for more
details on which additional props that are available.

<Props of="SimpleTable" />

## TableRow Props

SimpleTableRow forwards all unknown props to the base native `<tr>` component. See
[`<tr>` on MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/tr) for more
details on which additional props that are available.

# Slider

Sliders allow users to make selections from a range of values.

Sliders reflect a range of values along a bar, from which users may select a single value.
They are ideal for adjusting settings such as volume, brightness, or applying image filters.

export const DemoComponent = () => {
  const [standardSliderValue, setStandardSliderValue] = useState(0)

    </>
  )
}

## Basic usage

Without label

```typescript type=live
<Slider value={50} handleChange={() => {}} />
```

With label

```typescript type=live
<SliderField label="Slider field label" value={50} handleChange={() => {}} />
```

With unit label

```typescript type=live
<SliderField
  label="Slider field with unit label"
  value={50}
  handleChange={() => {}}
  unitLabel="Unit label"
/>
```

Slider with ticks

```typescript type=live
<Slider
  min={-10}
  max={10}
  value={0}
  handleChange={() => {}}
  tickConfig={{
    ticks: [{ position: -10 }, { position: 0 }, { position: 10 }],
  }}
/>
```

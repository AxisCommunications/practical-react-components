# Modal

The modal component provides a solid foundation for creating dialogs, popovers, lightboxes, or whatever else.

export const DemoComponent = ({}) => {
  const [open, setOpen, unsetOpen] = useBoolean(false)
  const [openSub, unsetOpenSub] = useBoolean(false)
  return (
    <>
      <Button label="Open modal" onClick={setOpen} variant="primary" />
      <Modal
        open={open}
        onClose={unsetOpen}
        titleId="modal-level-1"
        focusDialog={true}
        className="my-modal"
      >
        <Button label="Close modal" onClick={unsetOpen} variant="primary" />
      </Modal>
    </>
  )
}

[`<div>` on MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/div) for more
details on which additional props that are available.

# Content

## TextBlock

TextBlock is 40px height block with default Typography if you don't use caption property.
If you use TextBlock with caption property, you get lighter color caption with default Typography
under text and the block get 8px padding both top and bottom.

### Demo

<>
  <TextBlock text="Text block" />
  <TextBlock text="Text block" caption="This is caption" />
</>

### Basic usage

```typescript type=live
<>
  <TextBlock text="Text block" />
  <TextBlock text="Text block" caption="This is caption" />
</>
```

## ContentDivider

It shows an edge to edge divider line in a content (in Card or in Dialog)

### Demo

<Card>
  <CardContent>
    <ContentDivider />
  </CardContent>
</Card>

### Basic usage

```typescript type=live
<Card>
  <CardContent>
    <ContentDivider />
    <SpaceBlock variant={32} />
    // Use usual Divider if you need space to the side
    <SpaceBlock variant={16} />
    <Divider />
  </CardContent>
</Card>
```

## FormSection

FormSection is a wrapper that gives proper height (40px) title (optional) and
proper space (8px) between blocks to all its children.
It is possible to use only title if you don't give children.

### Demo

<>
  <FormSection title="FormSection group title">
    <TextInputField
      label="Text input field label"
      value=""
      onValueChange={() => {}}
      placeholder="Placeholder"
      width="large"
    />
    <TextInputField
      label="Text input field label"
      value=""
      onValueChange={() => {}}
      placeholder="Placeholder"
      width="large"
    />
  </FormSection>
</>

### Basic usage

```typescript type=live
<>
  <FormSection title="You can also use only title" />
```

## ContentListItem

List item for a content (in Card or in Dialog).
You can choose height between medium (40px) and large (48px).
Default is medium.

### Demo

<Card>
  <CardContent>
    <ContentListItem>
      <Typography>Medium content list item</Typography>
      <Switch value={false} onChange={() => {}} />
    </ContentListItem>
    <ContentDivider />
    <ContentListItem listHeight="large">
      <Typography>Large content list item</Typography>
      <Switch value={false} onChange={() => {}} />
    </ContentListItem>
    <ContentDivider />
    <ContentListItem>
      <RadioButton
        name=""
        value="me"
        label="Content list item"
        onValueChange={() => {}}
        partial={false}
        checked={true}

## ContentListItemWithHover

List item for special interaction with RadioButton.
The RadioButton gets hover effect even if the mouse is on the second content.

### Demo

export const RadioSelectableListItem = () => {
  const [hovered, setHovered] = useState(false)
  return (
    <ContentListItemWithHover onHover={setHovered}>
      <RadioButton
        name=""
        value="me"
        label="Content list item with hover"
        onValueChange={() => {}}
        partial={false}
        checked={true}
        hovered={hovered}
      />
      <Typography>Second content</Typography>
    </ContentListItemWithHover>
  )
}

<Card>
  <CardContent>
    <RadioSelectableListItem />
    <ContentDivider />
    <ContentListItem>
      <RadioButton
        name=""
        value="me"
        label="Content list item"
        onValueChange={() => {}}
        partial={false}
        checked={true}
      />
      <Typography>Second content</Typography>
    </ContentListItem>
  </CardContent>
</Card>

### Basic usage

```typescript
export const MyComponent = () => {
  const [hovered, setHovered] = useState(false)

  return (
    <Card>
      <CardContent>
        <ContentListItemWithHover onHover={setHovered}>
          <RadioButton
            name=""
            value="me"
            label="Content list item with hover"
            onValueChange={() => {}}
            partial={false}
            checked={true}
            hovered={hovered}
          />
          <Typography>Second content</Typography>
        </ContentListItemWithHover>
      </CardContent>
    </Card>
  )
}
```
